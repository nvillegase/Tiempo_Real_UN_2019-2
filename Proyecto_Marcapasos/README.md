### Instrucciones

Para ejecutar el código del proyecto, primero se debe correr (desde la carpeta del proyecto) el `Makefile` con el comando
```
make
```
Esto generará un archivo binario `main`. Para ejecutarlo, escribir en la terminal:
```
./main
```
